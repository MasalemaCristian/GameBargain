# GameBargain

Versión mínima del proyecto para mostrar ofertas de videojuegos. Estructura básica con Tailwind CSS.

Comandos útiles:

- Compilar CSS (genera `public/output.css`):
  `npm run build:css`
- Desarrollo (recompila automáticamente):
  `npm run dev`
- Servir la carpeta `public` localmente:
  `npm start`

Notas:
- Este repositorio está reducido a la estructura mínima. Ejecuta `npm install` para instalar dependencias antes de compilar.
